• A data source cat is declared using the <:data-source> directive.
  • The data is provided in JSON format by a local service at address /cat-facts as { "fact": "..." }.
  • A [['[[']]cat.content.fact]] expression is used to reflect received data in the page.
  • A button is added to refresh the data with the trigger() method.
Server Side Rendering is active by default
  • The server performs a first request and includes received data in the output page.
  • The client finds the initial request was already made and doesn't touch the DOM.
  • It can then perform further requests as needed.
